# CMPE 256 – Advanced Topics in Data Mining: Recommender Systems

**San José State University** | Spring 2026

## Course Overview

Graduate-level course covering the theory and practice of recommender systems, including:

- Data preprocessing and feature engineering
- Collaborative filtering and matrix factorization
- Deep neural networks for recommendations
- Graph mining for recommender systems
- Foundation models in recommendation pipelines
- Fairness and bias in ML
- Evaluation methodologies

## Repository Structure

